Question-2: What is the difference between innerHTML, innerText, and textContent?

Ans:
innerHTML:includes HTML tags.
innerText:Only visible text.
textContent:All text, including hidden.

Question-3: What is event delegation in the DOM?

Ans:
Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling, reducing multiple event listeners.

Question-4: What is event bubbling in the DOM?

Ans:
Event bubbling is a DOM mechanism where an event starts from the target element and moves up to its parents and activates their event handlers.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans:
Create:document.createElement("")
Add:parent.appendChild(child)/parent.append(child)
Remove:element.remove()/parent.removeChild(child)
